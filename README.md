# ask

## What is ask?
`ask` is a utility for Linux designed to make your experience smoother. It works like `man`, but in reverse!

## How to use ask?
Just type `ask` in your terminal, followed by your question, and you'll find the exact utility you need (sometimes!).

## Contributing
There are many ways to make this project better.
You can make the main program faster by editing the `ask.c` file, or add more commands to the `commands.json` file.

## How to install it?
Use provided script ( it's called `install` :D ).
